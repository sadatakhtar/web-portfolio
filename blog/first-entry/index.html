Today I downloded github-desktop for ubuntu which I previously did not have installed on my desktop. I have in the past used Gitkraken and other variants of a Git client but I have always preffered using Git via command line.

Using Git in the terminal is very easy but can seem daunting if you havent used it before. There are only a few commands which you need to familiarise yourself with, they are:

  • Git init - used to initialise git.
  • Git add "filename" - used once you have made changes to your code.
  • Git status - used to check the status.
  • Git commit -m "message here" - used to commit before uploading to github.
  • Git push -u origin master - used to upload to master branch on github.
  • Git pull - used to retrieve a repository from github to your local machine.
